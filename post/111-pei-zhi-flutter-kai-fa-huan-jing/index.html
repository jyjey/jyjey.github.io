1.1.1配置flutter开发环境

2020-05-17 2 min read

介绍

flutter使用的开发语言为dart，也就是说我们配置开发环境时要同时配置dart和flutter的开发环境

配制开发环境

1.下载flutter sdk

官方下载地址
注意不同的系统下载对应的SDK（我们以windows为例）我们下载最新的1.17.1-stable版本

解压后的 flutter sdk 目录

我们发现flutter sdk中已经包含了dart sdk

2.Windows下的环境配置

右键 此电脑>属性>高级系统设置>环境变量
（以下配置的环境变量可以为用户环境变量，也可以是系统环境变量，注意三个保持一致就可以了）
在Path变量中增加一条flutter的变量

由于flutter的爸爸是google由于某些原因国内访问可能会有问题
所以我们需要配置国内的镜像地址

FLUTTER_STORAGE_BASE_URL
https://storage.flutter-io.cn

PUB_HOSTED_URL
https://pub.flutter-io.cn



其他平台暂时不做说明，有需要请自行百度

3.配置开发工具（Android Studio、VS code）

Android Studio
File>Setting>Plugins 搜索flutter并安装（一般情况dart插件会一同安装，如果没有请继续搜索dart安装插件）安装完需要重启生效

VS code
见图

到此开发环境就已经配置完成了
